# GLB Studio — Real-Time 3D Modeler

A browser-based, real-time 3D modeling tool built with **pure WebGL** — no frameworks, no dependencies.

## Features
- Real-time PBR rendering (Metalness · Roughness · Emission)
- 6 primitive mesh types — Cube, Sphere, Cylinder, Cone, Torus, Plane
- Live scene hierarchy with multi-object support
- Transform controls — Position, Rotation, Scale
- Optimization panel — Auto LOD, Draco, KTX2, Mesh Merging
- Real-time stats — vertex count, triangle count, draw calls, GLB size
- Mobile-ready — touch orbit, pinch-to-zoom, frustum culling
- GLB export pipeline targeting sub-200KB assets

## How to Use
1. Download or clone this repository
2. Open `glb-modeler.html` in any modern browser
3. No installation or server required

## Tech Stack
- WebGL 2.0 (raw, no libraries)
- Vanilla JavaScript
- HTML5 + CSS3
